# 电
## 电线类型

![image](https://cdn.jsdmirror.com/gh/yanickxia/picx-images-hosting@master/image.26lvovyxsm.webp)

- BV：是一根铜丝的单芯线，比较硬，也叫硬线。
- BVR：是由多股铜丝绞在一起的单芯线，也叫软线。
- RV：也是软电线，是由很多股细铜丝绞在一起的单芯线，家装一般不用。
- BVVB:是两根以上BV线用护套套在一起的硬护套线。
- RVV：俗称软护套线，也就是2根、3根或者4根RV线用护套套在一起。


> 家庭装修推荐以 BV 为主，部分采用 BVR

## 电线标号
电线标号通常由 **特性代号 + 导体代号 + 绝缘代号 + 护套代号 + 额定电压 + 规格** 组成，格式如：
**WDZB-BYJ-450/750V 3×2.5mm²**

### **1. 导体材料（第1位）**
- **无字母**：默认铜芯（T），国标铜线通常省略
- **L**：铝芯（如：BLV 铝芯线）

### **2. 绝缘层材料（核心）**
- **V**：聚氯乙烯（PVC），最常见，性价比高
- **Y**：聚乙烯，绝缘性好但较硬
- **YJ**：交联聚乙烯，耐高温（105℃）、寿命长
- **X**：橡胶绝缘，柔韧性好

### **3. 线芯特征**
- **B**：固定布线用（硬线），单股实心铜芯
- **R**：软线，多股绞合铜芯，易弯折
- **S**：双绞线（如：RVS 双绞软线）

### **4. 护套层（有护套的电线）**
- **V**：聚氯乙烯护套（如：RVV）
- **Y**：聚乙烯护套

### **5. 特殊性能（前缀，最重要）**
- **ZR 或 Z**：阻燃，离火自熄
- **NH**：耐火，950℃火焰下可通电90分钟
- **WD**：低烟无卤（W-无卤，D-低烟）
- **WDZ**：低烟无卤阻燃（分A/B/C级，B级最常用）
- **WDZN**：低烟无卤阻燃耐火（最高安全等级）


对于家装来说，WDZB-BYJ 就是比较适合的电线，解释起来就是

| 字母段 | 含义 | 核心作用 |
|:--- |:--- |:--- |
| **WDZ** | 无卤低烟阻燃（Wu Lu Di Yan Zu Ran） | ✅ 燃烧时**不释放有毒卤气**、烟雾极少，逃生能见度高，无腐蚀性气体，避免二次伤害 |
| **B** | 阻燃等级为**B级** | ✅ 阻燃性能：离火自熄，阻止火焰蔓延，家装/工装最常用的阻燃等级（A级最高，B级性价比最优） |
| **BYJ** | 铜芯交联聚乙烯绝缘布电线 | ✅ 导体是纯铜，绝缘层为**交联聚乙烯（XLPE）**，耐高温、耐老化、绝缘性强，比传统BV线（聚氯乙烯）更耐用 |

# 注意事项

- 开竖槽：墙面开槽要以竖槽为主，有横槽建议不要超过30cm
- 强弱电不同槽，交叉处包锡纸
- 空调、冰箱等大功率电器单独设回路，后期断电检修不影响冰箱使用
- 普通照明/插座选2.5平方铜线、空调/冰箱、厨卫插座选4平方铜线

## 穿线管

家里电线管优先选 **PVC阻燃绝缘穿线管**，主流规格按管径（外径）分16mm、20mm、25mm，其中20mm是最常用的核心规格。
> 选 JG 3050-1998 国标阻燃 PVC 管，标注「阻燃 B 级」（ZR-B），管壁印字清晰有品牌 / 规格 / 阻燃标识。
### 主流规格及适用场景
1. 16mm（常用代号Φ16）：适用于单股2.5mm²及以下电线，比如普通照明回路、单个插座的支线，适合墙面暗埋的窄槽场景。
2. 20mm（Φ20）：家庭装修首选，适配2.5mm²多股线（如插座回路3根线）、4mm²单股线，通用性强，墙面、地面暗埋都适用。
3. 25mm（Φ25）：用于大功率回路，比如空调（4mm²及以上电线）、厨房/卫生间总回路，或同一管内穿线数量较多的情况。

### 关键选型要求
- 材质：必须选 **阻燃PVC管**（标注“阻燃”“绝缘”，符合国标GB/T 14823.1），避免非阻燃普通塑料管。
- 厚度：家用选中等厚度（管壁约1.2-1.5mm），工程常用加厚款，无需过度追求超厚，满足穿线和暗埋保护即可。
- 辅助规格：弱电（网线、电视线）建议用单独的16mm或20mm线管，与强电分管铺设，避免干扰。

### 安装注意要点
- 穿线率：同一线管内电线总截面积不超过管内截面积的40%，避免后期维修无法抽换电线。
- 弯曲：弯曲半径不小于管径的6倍，避免折扁影响穿线和绝缘性。

> 家庭装修推荐以 阻燃 PVC， 20mm 为主要规格，16mm 为辅，壁厚 1.5mm+

### 规格匹配
规格匹配（按电线平方选，核心！）：
✅ 国标要求：管内电线总截面积 ≤ 线管内截面积的 40%，剩余 60% 留散热 + 检修余量，WDZB-BYJ 电线耐高温但仍需散热，超量直接返工！

## 电线
### 主流线径及适用场景
1. 1.5mm²（BV/BVR型）：仅用于普通照明回路，比如客厅、卧室的吸顶灯、筒灯，单股或多股都适用，承载功率足够日常照明需求。
2. 2.5mm²（BV/BVR型）：家庭装修核心规格，适配所有插座回路（厨房、卫生间、客厅、卧室插座）、普通电器支线，能承载多数家用电器（如冰箱、电视、电脑）。
3. 4mm²（BV/BVR型）：用于大功率回路，比如空调（挂机、柜机）、厨房总进线、卫生间总进线，以及电热水器、烤箱等大功率电器专线。
4. 6mm²（BV/BVR型）：仅用于入户总电源线，或别墅、大平层的总进线回路，普通两居室、三居室一般用不到。

### 关键选型要求
- 材质：必须选铜芯线，导电性能好、发热少，避免铝芯线（易氧化、接头易松动）。
- 类型：家用选BV型（单股硬线）更易穿管、接线牢固；BVR型（多股软线）柔韧性好，适合转弯多的复杂布线，可按需选择。
- 标准：认准“国标”（符合GB/T 5023），包装上需标注线径、材质、额定电压（家用选450/750V）、生产厂家。
- 颜色：按规范区分（火线用红/黄/绿，零线用蓝，地线用黄绿双色），方便后期维修。

### 搭配注意事项
- 线径与线管匹配：1.5mm²、2.5mm²电线可穿16mm或20mm线管，4mm²电线建议穿20mm或25mm线管。
- 大功率电器专线：空调、电热水器、厨房大功率电器（如集成灶）需单独走回路，用4mm²电线+独立空开，避免过载。

### 速算表

| 电线线径（mm²） | 铜芯线类型 | 额定承载功率（W） | 核心适用场景 |
| :------------- | :--------- | :--------------- | :----------- |
| 1.5            | BV/BVR     | 1300-1600        | 普通照明回路（吸顶灯、筒灯） |
| 2.5            | BV/BVR     | 2200-2800        | 全屋插座回路、普通电器支线（冰箱、电视） |
| 4              | BV/BVR     | 3100-4000        | 空调、电热水器、厨房/卫生间总回路 |
| 6              | BV/BVR     | 4200-5500        | 入户总电源线、大平层/别墅分路总进线 |

### 关键备注
- 功率基于单相220V电压计算，实际承载会受环境温度、穿线数量影响（穿线越多、温度越高，功率需下调10%-20%）。
- 表格中为安全功率范围，避免长期满负荷运行，延长电线使用寿命。
- 大功率电器（如3匹柜机、即热式热水器）建议按上限匹配线径，且单独走回路。


## 水管

家用装修水管优先选 **PPR热熔管**，主流规格按外径分20mm（4分管）、25mm（6分管）、32mm（1寸管），其中25mm是最常用的主力规格，适配多数家庭用水场景。
### 主流规格及适用场景
1. 20mm（4分管，代号DN15）：
- 用于分支管线或小流量场景，比如单个水龙头、卫生间面盆、阳台水龙头的支线。
- 适合老房改造、空间受限的窄槽暗埋，或对水流需求不高的区域。
2. 25mm（6分管，代号DN20）：
- 家庭装修核心规格，全屋主力管线首选。
- 适配厨房总进水、卫生间总进水、热水器进出水、全屋主干管，水流比4分管大，满足同时使用多个电器（如洗衣机+水龙头）的需求。
3. 32mm（1寸管，代号DN25）：
- 用于入户总进水管、大平层/别墅的主干管，或大流量设备专线（如中央净水机、即热式热水器）。
- 普通两居室、三居室若水压正常，25mm主干管已足够，无需盲目选32mm。

### 主流规格-压力等级-壁厚对照表
| 水管外径（mm） | 压力等级（常用） | 壁厚（mm） | 核心用途 |
| :------------- | :--------------- | :--------- | :------- |
| 20（4分管）    | S3.2级（2.0MPa） | 2.8        | 冷水回路、分支管线 |
| 20（4分管）    | S2.5级（2.5MPa） | 3.4        | 热水回路、小流量热水支线 |
| 25（6分管）    | S3.2级（2.0MPa） | 3.5        | 冷水主干管、多数家庭冷水回路 |
| 25（6分管）    | S2.5级（2.5MPa） | 4.2        | 全屋热水主干管、厨房/卫生间热水回路（首选） |
| 32（1寸管）    | S3.2级（2.0MPa） | 4.4        | 大平层入户总进水管（冷水） |
| 32（1寸管）    | S2.5级（2.5MPa） | 5.4        | 中央热水系统主干管、大流量热水回路 |


PPR管明确分冷水和热水管，核心通过**压力等级、壁厚**区分，部分品牌会用颜色辅助标识，两者不能混用。
### 核心区分维度
1. **压力等级与壁厚**：
   - 热水管：常用S2.5级（2.5MPa），壁厚更厚（如25mm管径热水管壁厚4.2mm），耐压耐热性更强，可承受70℃长期热水输送。
   - 冷水管：常用S3.2级（2.0MPa），壁厚稍薄（如25mm管径冷水管壁厚3.5mm），仅适用于常温（≤40℃）冷水输送。
2. **外观标识**：
   - 热水管：表面会标注“Hot”“热水”或红色标识，部分品牌直接用红色管材区分。
   - 冷水管：标注“Cold”“冷水”或蓝色标识，管材多为白色、灰色。
3. **材质本质**：原料均为PPR，但热水管生产时配方更侧重耐高温稳定性，冷水管配方侧重常温耐压性。

### 关键使用原则
- 禁止混用：冷水管不能用于热水回路，高温下易老化、开裂漏水；热水管可替代冷水管使用（壁厚和耐压性更优），但性价比不高。
- 场景适配：家庭装修中，厨房、卫生间的热水龙头、热水器进出水、淋浴管线用热水管；冷水龙头、入户支管、阳台冷水管线用冷水管。
- 配件匹配：冷热水管的管件需对应（热水管件压力等级与热水管一致），避免“热水管配冷水管件”导致接口承压不足。

### 关键选型要求
- 材质：认准 **PPR管**（无规共聚聚丙烯），耐高温（可承受70℃热水）、耐腐蚀、热熔连接后密封性好，避免PVC管（不耐高温，不适合热水管）。
- 压力等级：家用选 **S2.5级（2.5MPa）** 或 **S3.2级（2.0MPa）**，热水管建议用S2.5级（管壁更厚，耐压耐热性更强）。
- 标准：符合国标GB/T 18742，包装标注材质、规格、压力等级、生产厂家，优先选品牌管材（如日丰、伟星等），配套管件（弯头、三通）需同品牌同规格。
- 冷热区分：热水管管壁更厚，部分品牌会标注“Hot”或红色标识，冷水管标注“Cold”或蓝色标识；热水管可替代冷水管使用，冷水管不能用于热水回路。

### 搭配注意事项
- 管径搭配：入户总进水管建议选25mm，分支管线用20mm，避免“大管接小管”导致水流损耗。
- 安装要求：PPR管热熔连接需专业操作，避免虚焊漏水；管道暗埋前需做压力测试（≥0.6MPa，保压30分钟无压降）。
- 配件匹配：管件规格需与水管一致（25mm水管配25mm弯头/三通），避免混用规格导致密封不严。
