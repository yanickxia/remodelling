                      地面#

                      瓷砖地面#

                      • 楼板：纯现浇钢筋混凝土楼板（无额外砂浆层），房屋最普遍的原始地面，本质是建筑承重核心层，没有附加砂浆
                      • 原始找平层：开发商会额外做5~10mm低标号水泥砂浆找平层，用来初步拉顺平整度，但这层不属于结构楼板本体，需要先排查判断是否保留：
                        • 无需铲除：用钥匙刮地面无掉粉、敲击无空闷声，找平层强度达标
                        • 必须铲除：刮地面掉砂粉、敲击有空鼓，后期会导致地砖空鼓脱层
                      • 二次找平层：用于贴砖前的找平工作
                      • 瓷砖粘结层：将瓷砖与找平层连接在一起的层
                      • 瓷砖面层：瓷砖本身

                      地暖地面#

                      • 楼板：是地面的结构基础，承担整个地面的承重作用，为上层地暖及装饰层提供稳定支撑。
                      • 地暖保温及盘管层：包含保温材料和地暖盘管，保温层减少热量向下散失，盘管通过热水循环实现地面供暖。
                      • 豆石混凝土填充层：包裹地暖盘管，固定管道位置，同时均匀传递热量，让地面温度更稳定。
                      • 二次找平：将填充层表面处理平整，为后续瓷砖粘结层提供光滑、均匀的基底，避免瓷砖空鼓。
                      • 瓷砖粘结层：用专用瓷砖胶将瓷砖与基层牢固粘结，确保瓷砖铺贴后不易脱落、移位。
                      • 瓷砖面层：是地面的装饰与保护表层，兼具美观、耐磨、易清洁的特点，同时保护内部地暖结构。

                      卫生间地面#

                      现在下沉式卫生间很多，这里单独列举出来

                      • 楼板：是卫生间地面的结构基础，承担地面所有层级的承重作用，提供稳定支撑。
                      • 基础防水层：是卫生间地面的第一道防水屏障，防止水渗透到楼板及楼下空间，避免渗漏问题。
                      • 陶粒回填层：填充卫生间沉箱（降板区域）的空间，同时陶粒质轻、透水，能缓冲地面压力并辅助排水。
                      • 二次防水层：在回填层上方再做一层防水，进一步强化防水效果，形成双重防水保障。
                      • 砂浆找平层：将二次防水层表面处理平整，为后续瓷砖粘结层提供光滑、均匀的基底，避免瓷砖空鼓。
                      • 瓷砖粘结层：用专用瓷砖胶将防滑瓷砖与基层牢固粘结，确保瓷砖铺贴后不易脱落、移位。
                      • 防滑瓷砖：是卫生间地面的装饰与功能表层，兼具防滑、耐磨、易清洁的特点，同时保护内部防水结构。

                      木板地面#

                      • 楼板：是木板地面的结构基础，承担地面所有层级的承重作用，提供稳定支撑。
                      • 原始砂浆找平层：对楼板表面进行初步找平处理，填补楼板的不平整，为后续层做基础铺垫。
                      • 二次找平层：进一步细化地面平整度，让基底更光滑均匀，确保后续防潮层、木地板的铺设效果。
                      • 防潮层：阻挡地面潮气向上渗透，避免木地板受潮变形、发霉，保护木地板的耐用性。
                      • 木地板：是地面的装饰与功能表层，兼具美观、脚感舒适的特点，同时起到保护内部层级的作用。

                      龙骨木板地面#

                      • 楼板：是木板地面的结构基础，承担地面所有层级的承重作用，提供稳定支撑。
                      • 找平层：将楼板表面处理平整，为后续防潮层、木龙骨的安装铺垫均匀的基底。
                      • 防潮层：阻挡地面潮气向上渗透，避免木龙骨、木地板受潮变形、发霉，保护木质构件的耐用性。
                      • 木龙骨：是木地板的支撑框架，通过固定在找平层上形成架空结构，既提升木地板的脚感弹性，也辅助分散承重。
                      • 木地板：是地面的装饰与功能表层，兼具美观、脚感舒适的特点，同时保护内部层级。

                      墙面#

                      瓷砖墙面#

                      这个墙面分层示意图展示了墙面装修的5个核心层级，每层功能各有侧重：

                      • 墙体基层：是墙面的“基础骨架”，通常是混凝土、砖墙等结构墙体，承担整个墙面的承重和稳定作用。
                      • 找平层：负责把基层不平整的表面处理得光滑、平整，为后续防水层、粘贴瓷砖打好基础，避免后续层出现空鼓、开裂。
                      • 防水层：主要用于卫生间、厨房等潮湿区域，能阻挡水渗透到墙体内部，防止墙体受潮、发霉，保护结构安全。
                      • 瓷砖粘结层：是瓷砖和墙面之间的“胶水”，通过专用瓷砖胶将瓷砖牢固粘贴在墙面上，确保瓷砖不会脱落。
                      • 瓷砖：是墙面的“表面装饰层”，既起到美观、易清洁的作用，也能进一步保护内部层级，提升墙面耐用性。

                      乳胶漆墙面#

                      • 墙体基层：是墙面的结构基础，通常为混凝土、砖墙等，承担墙面的承重与稳定性。
                      • 找平层：将基层不平整表面处理得光滑平整，为后续涂层铺垫，避免墙面出现凹凸、开裂。
                      • 防水层：多用于潮湿区域，可阻挡水分渗透，防止墙体受潮、发霉，保护内部结构安全。
                      • 腻子层：进一步细化墙面平整度，填补细微缝隙与孔洞，让墙面更光滑细腻，提升乳胶漆的附着效果。
                      • 乳胶漆层：是墙面的装饰与保护表层，既提供美观色彩，也具备防水、耐擦洗、防污功能，保护内部层级。

                      顶面#

                      乳胶漆顶面#

                      • 顶面楼板：是顶面的结构基础，承担顶面的承重作用，为上层装饰层提供稳定的支撑载体。
                      • 原始抹灰层：对楼板底面进行初步找平、找顺，填补楼板表面的不平整，为后续层做基础铺垫。
                      • 腻子层：进一步细化顶面平整度，填补细微缝隙和瑕疵，让顶面表面更光滑细腻，提升乳胶漆的附着效果。
                      • 乳胶漆面层：是顶面的装饰与保护表层，提供美观的视觉效果，同时具备耐脏、易清洁的特性，保护内部层级。

                      吊顶#

                      • 顶面楼板：是顶面的结构基础，承担整个吊顶系统的承重作用，为后续吊顶层提供稳定支撑。
                      • 原始抹灰层：对楼板底面进行初步找平处理，填补楼板表面的不平整，为轻钢龙骨的安装做基础铺垫。
                      • 轻钢龙骨架：是吊顶的支撑框架，通过连接件固定在楼板上，起到承重、塑形的作用，决定吊顶的造型与稳定性。
                      • 石膏板层：作为吊顶的基层面板，依附在轻钢龙骨上，构成吊顶的基本形态，同时具备一定的隔音、防火性能。
                      • 腻子层：细化石膏板表面的平整度，填补板缝、钉眼等瑕疵，让顶面更光滑细腻，提升乳胶漆的附着效果。
                      • 乳胶漆面层：是吊顶的装饰与保护表层，提供美观的视觉效果，同时具备耐脏、易清洁的特性，保护内部吊顶结构。
