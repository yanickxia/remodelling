                        墙面工序#

                        在 拆旧阶段 完成了墙面的拆除工作，继续来到墙面的恢复阶段。我们把视角拉回到拆旧完成后的那一刻。此时你面对的是一面“毛坯墙”（可能是红砖，可能是保留了老砂浆层的墙）。要把这面粗糙的墙变成最终细腻洁白的墙面，需要经历一个三步走的过程：找平（塑形） -> 批嵌（磨皮） -> 饰面（上妆）。

                        本章节覆盖瓦工，不涉及油漆

                        第一步：固化与底层找平 (The Muscle)#

                        1. 涂刷墙固 (界面剂)#

                        在 J/GJT 220-2010 抹灰砂浆技术规程 中，针对不同的墙面有描述，如果是混凝土墙体，需要此步骤 参考 [6.1.1内墙抹灰基层宜进行处理，并应符合下列规定]

                        （依据 JGJ/T 220-2010 标准）

                        如果您的墙面是现浇混凝土墙（表面光滑、甚至有脱模剂残留），且需要用水泥砂浆进行厚找平（抹灰），单纯滚涂液体墙固是不够的，挂不住重的水泥层。

                        • 标准工艺：JC/T 907-2018 界面剂 造毛
                        • 配方：使用专用的界面剂乳液 + 水泥 + 砂 搅拌成浆糊状（即界面砂浆），或者直接购买成品干粉界面砂浆。
                        • 操作：用滚筒或抹刀将界面砂浆涂抹在混凝土表面，形成粗糙的、带颗粒的表面（俗称拉毛）。
                        • 要求：根据规范，厚度不宜大于 2mm，必须覆盖全部基层，稍微收浆硬化后，再进行后续的抹灰作业。

                        对于烧结砖砌体的基层,应清除表面杂物、残留灰浆、舌头灰、尘土等,并应在抹灰前一天浇水润湿,水应渗人墙面内10mm~20mm。抹灰时,墙面不得有明水。

                        2. 粗找平 (Leveling)#

                        这一步是用找平石膏（或水泥砂浆）对墙面的凹凸不平进行填充。在这里，你需要做一个关乎预算和效果的重要决定：是做“顺平”还是“垂平”？

                        🅰️ 工艺选择：顺平 vs 垂平#

                        维度顺平 (Visual Leveling)垂平 / 冲筋找平 (Vertical Leveling)
                        定义“顺势而平”。以墙面原本的走势为基础，填补明显的坑洼。“绝对垂直”。先在墙上打“筋条”（基准线），把墙面强行修正成绝对垂直于地面的平面。
                        优点省钱、省空间。找平层比较薄，不会吃掉太多房间面积。精准、方正。墙角绝对90度，墙面绝对垂直。
                        缺点墙角可能不是90度，墙面可能不垂直。装柜子会有缝隙。贵（人工费+材料费高），厚（可能会让房间变窄1-2cm）。
                        适用场景全屋大部分区域（卧室床头、沙发背景、窗帘盒遮挡区）。推荐区域：① 安装定制柜的墙面（否则有缝）；② 厨房卫生间（贴砖需要）；③ 房门洞口（否则门套线有缝）；④ 踢脚线区域；⑤ 洗墙灯照射区域。

                        🅱️ 材料选择#

                        根据墙面基础不同，分为两种材料：

                        • 底层粉刷石膏 (Plastering Gypsum)：

                          • 适用：卧室/客厅干区、老房砂浆层表面的找平。
                          • 特点：推荐。粘结力极强，干得快，不易空鼓开裂，且厚度可控（从5mm到20mm都能做）。

                        • 水泥砂浆 (Cement Mortar)：

                          • 适用：厨卫湿区、新建墙体抹灰、拆除红砖后的修补。
                          • 特点：硬度高，耐水，但干得慢，收缩率大（易裂）。

                        垂平工艺：冲筋找平：工艺详解#

                        核心原理#

                        “筋”就是参照线（标尺）。先在墙上做出几条绝对垂直、厚度精准的“轨道”（即冲筋），等这些轨道干了变硬后，工人拿着长长的靠尺，架在两根轨道上进行批刮。这样刮出来的墙面，平整度完全取决于那两根轨道，从而消除了人为手抖的误差。

                        第一步：打点（做灰饼）—— 确立基准点#

                        在做“轨道”之前，必须先定“站点”。

                        1. 激光定位：用红外线水平仪打出垂直线，确定墙面最高点和最低点。
                        2. 做灰饼：根据激光线，在墙面的上、下、左、右角落，用石膏或砂浆点出几个小方块（厚度由找平厚度决定）。这些小方块就叫灰饼。

                        实践过程中，很多时候会跳过这一步

                        第二步：冲筋（做轨道）—— 连接基准线#
                        1. 连接：在上下两个灰饼之间，填满石膏或砂浆。
                        2. 压实成型：用长靠尺压在两个灰饼上，把中间填入的材料压平，形成一条竖向的、平整的石膏/砂浆带。这条带子干透后硬度很高，就是所谓的筋。
                        3. 间距控制：如果房间墙面高度 小于 3.5m（家装绝大多数情况），应该做立筋（竖向的），而且两根筋的间距不宜大于 1.5m。为什么要控制间距？ 因为刮平用的刮尺通常是 2米的。如果间距太大，刮尺够不着两头，就没法找平了。
                        第三步：批刮找平（填充）—— 填满路面#
                        1. 等待硬化：冲筋完成后，至少要等 2小时 后，才能开始抹底灰。这是为了让“筋”有足够的强度，不会被刮尺刮坏。
                        2. 填充刮平：在两根“筋”中间填充找平石膏。
                        3. 依靠尺刮平：工人拿着长刮尺，紧贴着左右两根“筋”，从下往上刮。多余的石膏被刮掉，不够的地方补上。因为刮尺是贴着硬“筋”走的，所以中间填出来的面也是绝对平的。
                        施工规范#
                        • 规范要求：冲筋应使用**“与抹灰层相同的砂浆”**。
                        • 实操避坑：如果大面积找平用的是粉刷石膏，那么冲筋也必须用粉刷石膏。千万不能用快干石膏（快粘粉）冲筋，虽然干得快，但硬度和收缩率不同，后期会有两条竖着的裂缝印子透出来。
                        验收标准#
                        • 工具：2米靠尺 + 塞尺。
                        • 方法：将靠尺任意角度贴在墙面上。
                        • 合格线：缝隙 ≤ 2mm（普通顺平工艺通常只能做到 ≤ 4mm）。

                        3. 养护环节 (Curing)#

                        找平层做完不是结束，水泥需要喝水才能变强。如果在这一步掉链子，前面的挂网、配比都白做了。

                        水泥砂浆 (Wet Areas)：必须洒水养护 原理：水泥硬化是“水化反应”，需要消耗水分。如果还没干透水分就挥发光了，水泥就会“烧坏”，导致起沙、开裂、强度不足。

                        🛠️ 水泥砂浆养护标准操作 (SOP)#

                        如果您的厨卫、阳台用了水泥砂浆找平，请监督工人执行以下操作：

                        1. 静置定型：刚抹完灰不能马上浇水，会把表面冲烂。通常需要阴干 24小时（视气温而定），指甲划上去只有浅痕时，才能开始养护。
                        2. 洒水保湿：工具：喷壶（推荐）或矿泉水瓶钻孔。不要用水管猛冲。频率：每天 2-3 次（早中晚）。保持墙面/地面处于湿润状态。周期：至少 3 天，最佳为 7 天。夏天水分挥发快，要增加频率。
                        3. 环境控制 (关窗)：在养护的前 3 天，必须关闭门窗 避坑：千万不能为了“散味”或“快干”而打开窗户穿堂风狂吹。风干速度 > 水化速度 = 必裂。

                        4. 水电开槽回填 (Slot Backfilling)#

                        水电工把管子埋好后，墙上全是“伤疤”（沟槽）。这些沟槽不能直接用找平石膏糊上去，必须单独处理，否则是后期开裂的重灾区。

                        • 红砖/混凝土墙：推荐用 水泥砂浆 回填。硬度高，能保护线管。
                        • 轻体墙/石膏板隔墙：推荐用 嵌缝石膏 回填。因为水泥收缩大，容易把轻体墙拉裂。

                        标准工艺#

                        1. 管卡固定：确保线管在槽内固定牢固，没有弹出来，且低于墙面至少 1cm（留出填灰空间）。
                        2. 润湿：回填前，向槽口喷水湿润（如果是水泥回填）。
                        3. 填充：填实，严禁空鼓。
                        4. 挂网：这是必须的！回填干透后，在槽口表面贴 玻纤网格布 或 牛皮纸，宽度要覆盖槽口两边各 10cm。

                        5. 门窗洞口修正 (Opening Correction)#

                        很多瓦工只管大面墙平不平，不管门洞直不直。结果全屋定制复尺时发现：门洞是梯形的，导致门套线装不上，或者缝隙巨大。

                        • 门套安装：室内门（木门/极简门）对门洞的垂直度要求极高。如果门洞上宽下窄，门套就装不进去；如果墙体厚度不一致，门套线就会一边贴墙一边翘起。
                        • 窗户安装：窗洞不正，窗框周围打胶就不严密，容易漏水。

                        在动手之前，必须明确我们要把洞口修成什么样：

                        • 垂直度：左右两侧墙体必须绝对垂直，误差 ≤ 3mm。
                        • 厚度一致：墙体的厚度（从左到右，从上到下）必须一致，误差 ≤ 3mm。
                        • 宽度标准：门洞上、中、下的宽度一致，不能是梯形。

                        整个过程类似于冲筋找平，不多描述

                        📝 验收作业表#

                        验收项目核心关注点工具合格标准 (及格线)优秀标准 (满分线)
                        墙固/界面剂附着力强光手电，指甲/硬币无漏刷、无流挂。颜色均匀，干透后指甲用力划不整块剥落。渗透极好。硬币都难以刮下，且表面有明显的粗糙感（造毛工艺）。
                        水电回填防裂肉眼，空鼓锤无裂缝。回填处与旧墙齐平。有挂网。表面覆盖了玻纤网/牛皮纸，且覆盖宽度单边 ≥10cm。
                        挂网工艺位置肉眼关键位置有网。新旧墙交接处、异质墙体连接处能看到网格痕迹。包裹严实。网被完全“夹”在找平层中间，不外露，不空鼓。
                        粗找平 (顺平)平整度2米靠尺，塞尺缝隙 ≤ 4mm。无大面积空鼓（空鼓锤敲击无咚咚声）。缝隙 ≤ 3mm。墙面平整，为后续腻子层省料。
                        冲筋找平垂直度2米靠尺，塞尺缝隙 ≤ 2mm。必须上靠尺测，不能只用眼看。缝隙 ≤ 1mm。能满足极简柜体、洗墙灯的严苛要求。
                        门窗洞口方正度红外水平仪，卷尺垂直误差 ≤ 3mm。上中下宽度一致，无梯形洞。墙厚一致。四个角的墙体厚度误差极小，门套线安装后严丝合缝。
                        养护 (仅水泥)硬度手掌，喷壶表面湿润（养护期内）。干燥后手搓不起沙、不掉灰。坚硬如石。表面呈青灰色，硬度极高，无任何细微干裂纹。

                        第二步：精细找平 (The Skin)#

                        这是墙面最关键的“磨皮”环节，决定了墙面平不平、细不细。下面的工序参考

                        • GB 50327-2001 住宅装饰装修工程施工规范
                        • JGJ/T 29 建筑涂饰工程施工及验收规程

                        不过几乎没有特别明确的施工标准

                        1. 阴阳角处理（Corners）#

                        • 工具：阴阳角条（PVC塑料条）。
                        • 工艺：在墙角的凹角（阴角）和凸角（阳角）预埋塑料条。
                        • 作用：让墙角变得笔直（90度垂直），并且保护阳角防磕碰。

                        验收#

                        • 核心指标：“方正”与“顺直”。
                        • 操作方法：
                          • 测方正：拿出阴阳角尺，直接卡在墙角。标准：指针误差应在 ±3mm 以内。如果是安装柜子的地方，要求 ±2mm 以内。
                          • 测顺直：站在离墙角2米远的地方，眯起眼睛看那条线直不直。或者用红外线水平仪打一条垂线对比。标准：肉眼看无明显弯曲，线条挺拔。

                        2. 批刮石膏（Plaster）#

                        一般市场上有三次批光的说法，这里可能是由 批刮石膏 + 批刮腻子 组合的。

                        • 大多数的水泥砂浆粗找平环境可能会批刮石膏环节
                        • 如果是用石膏粗找平，就没有这个过程
                        • 石膏找平 vs 批刮腻子：这是前后承接的关系。
                        • 先用石膏把墙面的大坑填平（粗糙面），再用腻子把表面收光（光滑面）。

                        验收#

                        • 核心指标：“牢固”与“大致平整”。
                        • 操作方法：
                          • 查空鼓：用空鼓锤轻轻敲击石膏找平层，尤其是新旧墙交接处、开槽回填处。 标准：不得有空鼓声，不得有裂纹。如有空鼓，必须铲掉重做，严禁直接刮腻子覆盖。
                          • 测平整：用 2米靠尺 紧贴墙面。标准：塞尺缝隙 ≤ 4mm。如果不平整度太大，后面刮腻子太厚会开裂。

                        3. 批刮腻子 (Puttying)#

                        通常需要刮 2-3 遍。

                        • 第一遍：主要为了找平，可能会厚一点。
                        • 第二/三遍：为了收光，让表面细腻。
                        • 材料选择：尽量选耐水腻子（N型）。
                        • 复习：拒绝“双飞粉+胶水”的现场调制腻子，推荐耐水腻子（执行标准 JG/T 298-2010 N型）。

                        验收#

                        • 操作方法：
                          • 抠墙角：用手指甲在阴阳角或墙面边缘用力抠一下。标准：耐水腻子干透后非常硬，指甲抠不动，没有明显的粉末掉落。如果一抠就掉粉，说明腻子没干透或者胶水含量不够（买到假货）。
                          • 看表面：检查表面是否有气泡、蜂窝孔。标准：表面应洁净、平实，无大面积砂眼。

                        4. 打磨 (Sanding)#

                        • 时机：腻子刮完半干不干的时候（通常是最后一遍腻子刮完2-3天内）。
                        • 工艺：工人拿着砂纸（配合灯泡侧光照射）打磨墙面。
                        • 重要性：“三分批，七分磨”。墙面平不平、有没有砂痕，全看这一步。打磨完后墙面会产生大量粉尘，必须清扫干净才能刷漆。

                        验收#

                        • 核心指标：“平整”与“无痕”。
                        • 操作方法：
                          • 侧光检查 (必做)：拉上窗帘，打开强光手电，紧贴墙面侧向照射。标准：墙面没有明显的波浪纹（凹凸不平），也没有明显的砂纸痕（划痕）。侧光下允许有极轻微的起伏，但不能有坑。
                          • 手摸触感：用手掌在墙面上摸一圈。 标准：手感光滑细腻，无颗粒感，且手上没有过多的浮尘（说明打磨后清扫到位）。
                          • 终极平整度：再次请出 2米靠尺。标准：顺平工艺：靠尺缝隙 ≤ 3mm。垂平/冲筋工艺：靠尺缝隙 ≤ 2mm（甚至更低）。

                        特别关注：挂网#

                        1. 挂什么网？(材料选择)#

                        根据材质和强度的不同，分为两种：

                        网的类型学名俗称硬度适用场景 (必背)

                        2. “全屋挂网”是智商税吗？#

                        装修公司经常忽悠业主：“老房子墙不行，必须全屋挂网，不然全裂，加钱 20元/平。”

                        • 什么时候也是必须的？

                          • 老房铲墙后：如果铲掉大白后，发现里面的砂浆层虽然没空鼓，但裂纹非常多（像龟背一样），这时候建议全屋挂玻纤网格布兜底。
                          • 保温墙：如果墙面有内保温层（泡沫板），必须全屋挂网，否则甚至挂不住腻子。

                        • 什么时候是冤大头？

                          • 新房/墙面状态良好：如果基层很结实，没有裂纹，完全没必要全屋挂网。挂了也不会增加墙体强度，纯属浪费钱。

                        3. 施工工艺#

                        很多人以为挂网是先把网钉在墙上，再抹灰，其实分情况：

                        • 钢丝网 (配合水泥)： 正确做法：先钉在墙上，或者先抹一层薄灰，把网压进去，再抹面层灰。关键是网要被包裹在水泥层中间，不能贴着墙皮（那样抓不住力）。

                        • 网格布 (配合石膏/腻子)：

                          • 正确做法：先批一层石膏，趁湿把网格布用抹刀压进去，再批第二层。
                          • 错误做法：干挂。直接把网格布贴在干墙上，然后外面刮腻子。这样网和墙是分离的，起不到抗裂作用。
