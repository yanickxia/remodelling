                硬装阶段

                硬装阶段最复杂的就是如何看懂报价，这里大致上分为几个部分。

                1. 基础拆改模块#

                - 拆旧：要看是按拆除体积还是按平米计价，重点确认是否包含拆后垃圾清运，很多低价报价不含清运，后期单独收200~500元/车，还要确认是否高空作业的额外收费；
- 新建墙体：要写清楚材质，红砖（120~160元/㎡）、轻体砖（100~130元/㎡）、石膏板隔墙（80~110元/㎡）价差很大，同时确认是否包含挂网、抹灰、后期门垛砌筑，这些都是常见漏项加价点。

                2. 水电工程（增项重灾区）#

                - 点位数量：包干价要明确包含多少个插座、开关、灯具、下水点位，超出点位每个收费多少（一般50~100元/个）；
- 材料细节：必须写清楚电线品牌型号（比如国标BV2.5平方铜线、国标BV4平方铜线）、水管品牌（伟星、日丰、保利），强弱电交叉是否包含防干扰锡箔纸包裹、下水管隔音棉是否赠送、卫生间等电位是否包含在内；
- 隐藏收费：水电开槽开混凝土墙一般要额外加10~15元/米，很多低价报价只算砖墙开槽；包管费按米计价150~220元/米，很多报价不含但必须做不然下水噪音大。

                3. 瓦工工程#

                - 瓷砖铺贴：要确认是否包含瓷砖背胶涂刷、拉毛处理，很多低价报价不含背胶，后期加价10~15元/㎡；还要看特殊工艺收费标准：通铺对缝、海棠角倒角、墙面薄贴加价都要写清楚；
- 防水：要写清楚防水品牌（东方雨虹、德高）、刷涂高度，淋浴区墙面必须刷到1.8米，普通卫生间1.2米，地面全屋满刷，闭水试验48小时要包含在报价里；
- 辅助项目：地漏、蹲便器安装、门槛石铺贴很多低价报价不含，收费50~200元/个。

                4. 木工工程#

                - 吊顶：区分双眼皮吊顶、边吊、无主灯悬浮吊顶的单价，确认是否包含龙骨品牌（轻钢龙骨还是木龙骨，轻钢更稳定）、隔音棉填充、收口条；
- 定制柜体：投影面积报价要写清楚包含板材品牌（ENF级颗粒板还是多层板）、五金（铰链、拉手）、抽屉数量，一般包含2~3个抽屉，加抽屉需额外收费100~200元/个；展开面积报价要核对每一片板材的用量和单价，避免虚增；
- 窗帘盒、垭口套是单独计价，要确认是否包含油漆上色。

                5. 油工工程#

                - 基层处理：要确认是否包含旧墙皮铲除、裂缝挂网、墙面找平，很多低价报价只包含刷乳胶漆，基层处理单独收费20~30元/㎡；
- 乳胶漆：要写清楚品牌系列（比如立邦净味120、多乐士竹炭清新居），报价按几遍腻子、几遍底漆、几遍面漆来算；
- 特殊收费：墙面找方、阴阳角条找直很多报价不含，单独收费10~15元/米。

                多大多数的单项都是基于 主材 + 辅材 + 人工 构成，其中

                • 人工 是一个相对不透明的部分，建议参考小红书，或者 闲鱼*120%
                • 主材 是一个非常透明的部分，无论是腻子，油漆，都可以从网上查询到
                • 辅材 一般不是大额，大致上没有太多差异都可以
