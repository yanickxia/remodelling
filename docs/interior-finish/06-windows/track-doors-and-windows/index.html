                        移门系统

                        • 移门系统 (Door Systems)
                          • 吊轨系统 (Top-Hung Systems)
                          • 地轨系统 (Dual Track Systems)

                        移门系统 (Door Systems)#

                        移门系统的核心区别在于谁受力（承重在顶上还是地上）以及轨道在哪里。不同的系统对硬装（墙面平整度、地面找平、木工吊顶）有着截然不同的要求。

                        吊轨系统 (Top-Hung Systems)#

                        核心特征：“上承重，下无轨”。 所有的重量都由门洞上方的横梁或墙体承担，地面只有极小的定位件（止摆轮），没有贯穿的轨道。下面的细分更多的只是五金的区别

                        1. 普通吊轨 (The Standard)#

                        —— 粗放型、工业感、外露

                        • 轨道形态：通常是一个比较宽大的U型铝合金槽（像个盒子）。
                        • 安装方式：
                          • 明装：直接拧在门洞上方的墙上或梁上，外面再扣一个装饰盖板（轨道盒）来遮丑。
                          • 半隐藏：虽然装在窗帘盒里，但抬头依然能看到那个巨大的金属槽和轮子挂件。
                        • 五金结构：滑轮通常比较大，挂在门顶，有时候能从缝隙里看到轮子。
                        • 槽点：
                          • 缝隙大：为了容纳粗糙的五金，门顶和轨道之间缝隙较大（漏光、漏音）。
                          • 晃动：老式吊轨的止摆轮比较简陋，门推起来容易“飘”。
                        • 适用场景：厨房、干区隔断（预算有限，且对颜值要求不极端的区域）。

                        2. 完美系统 / 嵌入式吊轨 (The Minimalist)#

                        —— 极简型、内嵌式、隐形

                        这是吊轨的“2.0 进化版”，核心逻辑是把能藏的都藏起来。

                        • 轨道形态：极窄、极薄的轨道。
                        • 安装方式 (木工介入)：
                          • 嵌入式：轨道不是“挂”在梁下的，而是**“埋”进吊顶里的**。装修完看，天花板上只有一条极细的黑色缝隙（约 1-2cm宽），轨道隐身了。
                        • 五金结构：滑轮和阻尼器被压缩到了极致，完全藏在极窄的门框或细缝里。
                          • 优点：颜值极高，几乎没有卫生死角。
                          • 代价
                            • 维修难：轨道嵌在吊顶里，五金坏了很难拆卸。
                            • 造价高：需要木工（开槽）、瓦工（地面找平）高度配合。
                        选购指南#

                        完美系统的核心在于**“三分看框，七分看轨”**。市面上所谓的“完美系统”大多通用 OPK (欧派克) 的五金方案，选购时请死磕以下参数：

                        1. 型材规格 (Profile Specs)

                        • 主流代号：市面上常说的 “4013” 或 “4010” 是指型材截面的尺寸。
                          • 40：指型材的深度（40mm），决定了门的稳固性。
                          • 13/10：指型材的可视面宽度（13mm或10mm），也就是正面看过去的那个极窄边框。
                        • 避坑：不要只追求“极窄”，边框过细（<10mm）会导致玻璃晃动。推荐 13mm-16mm 的可视面宽度，视觉极简且结构稳定。
                        • 壁厚：型材铝壁厚度建议 ≥ 2.0mm，以保证门扇高大时不变形。

                        2. 表面工艺 (Surface Finish)

                        • 首选：瓷泳 / 电泳 (Porcelain Electrophoresis)。手感像肌肤一样细腻，抗指纹，不褪色，质感极级棒。
                        • 次选：氟碳喷涂。颗粒感较强，耐磨性尚可。
                        • 不推荐：普通喷涂。容易掉漆，质感廉价。

                        3. 轨道系统 (Track System) —— 核心中的核心 目前市面上 90% 的完美系统都在用 OPK (欧派克) 的方案，它的轨道分为不同高度：

                        • 低轨 (3.5cm)：对吊顶空间要求低，但轮子较小，承重和顺滑度一般。
                        • 中轨 (4.5cm) / 高轨 (6.0cm)：强烈推荐选高不选低。
                        • 原理：轨道越深（越高），能容纳的滑轮直径就越大。大轮子滚动的平稳性、静音性远超小轮子，且防跳轨能力更强。
                        • 施工要求：需木工吊顶预留更深的槽（≥6cm），装修前务必确认吊顶高度。

                        4. 阻尼器 (Damper)

                        • 标配：双向阻尼 (Two-way Soft Close)。 作用：无论开门还是关门，推到尽头时都会自动减速吸合。既能防止门大力撞墙，又能防止夹手。如果商家配的是单向阻尼（只有关门缓冲），请要求升级。

                        5. 玻璃配置 (Glass)

                        • 厚度：标准配置 8mm 钢化玻璃（必须有 CCC 认证）。超高门（>2.4m）建议升级为 10mm 或夹胶玻璃。
                        • 颜色/款式：
                          • 超白玻：通透干净，不泛绿（推荐）。
                          • 灰玻/茶玻：有高级感，适合衣帽间。
                          • 油砂/玉砂：透光不透人，适合卫生间。
                          • 长虹/瓦楞：颜值高，但价格通常要加钱。
                          • 避坑：尽量避免复杂的格栅造型，积灰难打理，且破坏极简感。

                        6. 移动扇工艺 (Sash Quality)

                        • 壁厚：必须 ≥ 2.0mm。若门高 > 2.4m，建议升级至 2.5mm 以防推拉颤动。
                        • 拉手：首选 CNC 一体精雕拉手（在型材上直接雕刻凹槽），拒绝拼接或粘贴拉手，追求极致的一体感。
                        • 表面：首选瓷泳/电泳（细腻抗指纹），次选氟碳喷涂，拒绝普通喷涂。

                        7. N+1

                        一般完美系统还有 N + 1 的说法，一般指的是，口袋门是否可以空出来一块就是 0

                        3. 幽灵门 (Ghost Door / Side-Hung)#

                        —— 悬浮感、侧挂式、网红款

                        这是吊轨的一个特殊变种，核心逻辑是轨随门动。

                        • 轨道形态：轨道安装在门扇背面，滑轮安装在墙上。

                        • 视觉效果：“只见门板不见轨”。门板像悬浮在墙面上滑动，极其炫酷。

                        • 施工死穴 (关键!)：

                        • 墙面垂平：必须做冲筋找平！ 因为门板紧贴墙面滑动（缝隙仅 ~5mm），墙面稍有不平就会摩擦门板。

                        • 踢脚线：门滑动的区域严禁安装明装踢脚线，必须做内嵌或断开处理。

                        • 适用场景：极简风格的卧室门、书房门、隐形门。

                        4. 谷仓门 (Barn Door)#

                        —— 工业风网红、装饰性强、密封“漏勺” 这是最具辨识度的明装吊轨。

                        • 轨道形态：完全外露。一根黑色（或金色）的金属长杆直接钉在门洞上方的墙面上，复古滑轮挂在杆子上。
                        • 优点：
                          • 装饰性极强：颜值高，适合美式、工业风、北欧风。
                          • 安装简单：不需要做门套，也不需要改动门洞结构，直接在墙上打孔就能装。
                        • 缺点：
                          • 密封性极差：为了保证门能滑动，门板离墙面必须留出 1-1.5cm 的缝隙。
                          • 隔音/隔味为零：几乎不隔音，也不隔油烟。
                        • 适用场景：衣帽间、储物间、洗衣房。
                        • 禁区：严禁用于卧室（不隔音）、卫生间（没法锁且漏气味）、厨房（漏油烟）。

                        地轨系统 (Dual Track Systems)#

                        核心特征：“下承重，上导向”。 所有的重量都由地面的轨道承担，顶部的轨道只负责防止门倒下来。这是最稳固的方案。

                        1. 传统凸轨 (The Old School)#

                        —— 已淘汰

                        • 轨道形态：地面粘一条凸起的金属轨道。
                        • 缺点：绊脚、积灰难清理、轮子容易卡头发。家里有老人小孩绝对不要装。

                        2. 极简地轨#

                        —— 稳如泰山、密封之王

                        • 轨道形态：
                          • 铜条/不锈钢条：直接在地砖上开极细的浅槽，嵌入实心金属条（仅凸出地面 1mm）。
                          • 嵌入式轨道：铺地砖时预埋 U 型轨道，与地砖齐平。
                        • 优点
                          • 承重无上限：适合 3米高的超大玻璃门。
                          • 不晃动：上下卡死，抗风压能力极强。
                          • 好打理：扫地机器人无感通过。
                        • 适用场景：阳台封窗移门（抗风）、卧室/书房（隔音密封）、超宽玻璃隔断。
