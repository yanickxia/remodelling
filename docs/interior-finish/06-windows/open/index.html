                开启扇

                门窗的开启扇对于窗户的影响也很大，这里单独介绍

                平开扇#

                平开门是指合页(铰链)装于门侧面、向内或向外开启的门。

                平开是最为成熟的结构，低楼层可以选择 外开，高楼层可以选择 内开，平开窗因为可以使用胶条进行密封，气密性非常的好

                内开#

                向内开的，就是内开

                外开#

                向外开的，就是外开

                内倒#

                可以从上面打开一条十厘米左右的缝隙，打开的部分悬于室内。

                上悬#

                通过五金件的不同连接方式实现外开或上悬；窗户上悬时，可以15—30的角度打开一定的缝隙，开启的部分悬在（室外）空中。

                推拉窗#

                普通推拉门以毛条来封闭，会有很差的气密性

                毛条最多挡挡灰，并没什么气密性可言，为了解决气密性，这里有几个几个变种
