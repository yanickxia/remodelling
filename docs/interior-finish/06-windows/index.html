                    门窗结构#

                    玻璃固定结构#

                    闭口选择：闭口针对的是窗户的稳定性，如果外部风压较大，需要选择闭口。这里判断的条件其实就是看是否有横向的支持筋。

                    密封结构#

                    密封结构决定的保温性

                    这里是 等压密封条，选择 EPDM 材料，分为 普通 发泡 软硬共济 三种。在接触面最好选择 软硬共济

                    工艺上要注意密封接触面越大越好

                    直接抄答案

                    因为密封是四周的，还要考虑转角的问题。

                    选择 1 > 2 > 3

                    墙面固定结构#

                    这里分为： 型材和型材之间的固定 和 型材和墙之间的固定

                    型材和型材之间的固定#

                    避坑 螺丝固定

                    型材和墙之间的固定#

                    这里需要关注施工和材料

                    1. 固定螺丝避坑小黄鱼

                    2. 螺丝不要少打

                    3. 结构胶关注抗位移

                    4. 发泡胶选择

                    在宽框体需要注意一个点，就是因为安装角码，不要是一个大空腔，会导致注胶不稳定
