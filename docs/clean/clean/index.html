                        拆旧阶段#

                        在拆旧阶段，我们需要先了解一下我们的房屋结构，这会帮助我们更好地了解施工报价。可以再复习一下 基础分层

                        基本拆除#

                        对于旧屋有一些比如床，桌子，既有的柜子等，会有一个旧房全屋拆除的步骤，会将家里的大部分物件进行拆解，然后打包好进行清运。

                        墙面拆除#

                        铲墙深度每层的价格差异很大，只铲掉饰面层重刷乳胶漆最便宜，铲到腻子层其次，铲到砂浆层价格最高，要根据自家墙面老化情况选择铲墙深度，避免被装修方过度加价。

                        对于老旧小区，一般需要拆到墙体，对于新房，一般需要拆到砂浆层。

                        注意：拆掉的越多，后面新建的越多，尽量减少拆除的内容。

                        在此阶段需要注意的事项是

                        • 拆除部分物件是可以回收的，需要约定好归属，比如门窗等。
                        • 拆除谈全包价，一口价，拆除+清运，清运费是常见的后期增项。
                        • 报价可以多参考几家，选择一个比较适合的价格。
                        • 对于外墙面还有外墙涂料/外墙漆，这部分拆除的价格会贵一些。

                        铲墙皮/腻子#

                        铲除腻子的具体操作流程

                        情况 A：普通腻子（非耐水腻子）—— 湿铲法#

                        这是老房最常见的情况，操作相对简单，也是默认报价中的操作方式。

                        1. 滚水浸泡：工人使用滚筒刷沾水，将墙面完全打湿。
                        2. 等待软化：等待几分钟，水分渗透进腻子层后，腻子会变软、起泡或粉化。
                        3. 手工铲除：工人使用铲刀，像削果皮一样将软化的腻子层成片铲下。
                        4. 露出底层：直到露出灰色的水泥砂浆层（或石膏找平层）为止。

                        情况 B：耐水腻子 —— 干磨法/机器铲#

                        如果是比较新的房子，或者开发商交付标准较好，使用的是耐水腻子，水泡不软，铲刀铲不动。

                        1. 直接打磨：需要使用墙面打磨机（自带吸尘功能的专业机器）或者角磨机进行深度打磨。
                        2. 物理破坏：靠机器的物理旋转力将腻子层磨掉。
                        3. 难点：这种方式灰尘极大、人工费更贵、且耗时更长。

                        铲砂浆#

                        如果说铲腻子是“皮肤护理”，铲砂浆层就是“刮骨疗毒”。

                        什么时候必须铲砂浆？#

                        并不是所有房子都需要铲到这一层，因为这非常费钱：

                        在墙面结构中，红砖/混凝土是骨架，砂浆层是肌肉，腻子乳胶漆是皮肤。正常情况下它们应该紧紧粘在一起。一旦中间有了缝隙，敲击时声音就会通过空气共振放大，发出“咚咚咚”的空响，所以叫“空鼓”。

                        • 大面积空鼓：拿空鼓锤敲击墙面，如果听到大面积“咚咚咚”的空洞声，说明砂浆层和砖墙已经分离，必须铲掉重做。
                        • 严重起沙：用手或钥匙用力划一下砂浆层，如果哗啦啦掉沙子，说明水泥标号不够或老化，贴砖挂不住，刷漆会开裂，必须铲。
                        • 受潮发霉/泛碱：墙面如果有严重的“白毛”或粉化，说明砂浆层已经废了。
                        • 全屋找平太差：如果墙面歪斜极其严重，有时候为了重新找直，装修公司会建议铲掉旧的，重新做冲筋找平。

                        为什么会空鼓#

                        空鼓是老房最常见的“顽疾”，通常由以下原因导致，无法通过修补表面解决：

                        1. 抢水现象：当年施工时砖墙过干，吸走了水泥里的水分，导致粘接失效。
                        2. 基层不净：墙面浮灰或油污未清理，导致“假粘”。
                        3. 结构老化：房屋多年的自然沉降和热胀冷缩，将原本贴合的砂浆层“震”开了。

                        标准施工步骤#

                        1. 水电定位与保护：这是最重要的一步！老房墙体里可能埋有旧的水管和电线管。工人必须先判断大概走向（或者直接断水断电），防止电镐打穿水管导致漏水。
                        2. 切割分界：如果有不需要铲除的墙面（比如和邻居的共用墙连接处），先用角磨机切出一道沟槽作为“止损线”，避免震动导致好墙皮开裂。
                        3. 电镐剔凿：工人手持电镐，从上往下，或者从空鼓处开始，将水泥砂浆层成块地凿下来。
                        • 目标状态：要看到内部的红砖（老房）或加气块/混凝土（新房）。

                        拆多少#

                        • 局部铲除：适用场景：房屋基础较好，只有零星的空鼓。

                          • 判断标准：单面墙体空鼓面积 < 20%。
                          • 操作方式：只把空鼓的那一块敲掉，而且不能只敲空鼓的范围，要向外多扩拆 5-10cm，确保边缘是牢固的。
                          • 优点：省钱。
                          • 缺点：新旧砂浆交接处容易因为收缩率不同产生裂缝（必须挂钢丝网来解决）。

                        • 整面/全屋铲除：适用场景：老破小、基础极差的安置房、或者打算贴大瓷砖的区域。

                          • 判断标准： 空鼓率过高：单面墙体空鼓面积 > 20%-30%。这时候补丁太多，不如重做划算且牢固。
                          • 严重起沙：手搓掉沙严重，说明整个墙面的水泥标号都不够，局部修补没有意义，根基烂了。
                          • 厨房卫生间：强烈建议全铲。因为这两处要贴瓷砖（尤其是现在流行的大板砖），如果基层不牢，瓷砖会连着砂浆层一起掉下来，安全隐患极大。
                          • 老房红砖墙：如果是80、90年代的老房，砂浆层通常已经风化酥松，建议全铲。

                        施工红线与避坑#

                        如果在做 局部铲除 ，必须注意以下工艺，否则后期必裂：

                        1. 切割方正：铲除空鼓时，不能敲成不规则的锯齿状。必须用切割机切成方方正正的矩形，这样后期填补砂浆时受力才均匀。
                        2. 新旧交接挂网：这是防止开裂的核心！
                          1. 在补新砂浆之前，必须在铲除区域的边缘（新旧交界处）挂钢丝网。
                          2. 网要覆盖旧墙至少 10cm，覆盖新填补区至少 10cm，像“创可贴”一样把裂缝隐患拉住。
                        3. 浇水湿润：补砂浆前，必须把老墙周围打湿，防止老墙吸走新砂浆的水分（就是之前说的“抢水”原理）。

                        顶面拆除#

                        墙顶和墙面的结构是相似的，拆除时需要注意的事项是一般都是拆到砂浆，一般口径里面的 墙皮 包含了 腻子 + 饰面。

                        地面拆除#

                        瓷砖地面#

                        拆旧瓷砖时，核心是铲除瓷砖本身及瓷砖下方的粘结层，部分情况还需处理基层，通常需要涉及 3 层左右，具体取决于原瓷砖的铺贴工艺。

                        • 必铲的核心层（2 层） 这两层是拆旧瓷砖的主要目标，必须彻底清除，否则会影响新瓷砖铺贴质量。

                          • 瓷砖面层：最外层的旧瓷砖本身，是视觉可见的部分，需要用工具将其从墙面或地面剥离。
                          • 瓷砖粘结层：瓷砖与基层之间的粘结材料层，常见的有水泥砂浆层或瓷砖胶层。这一层必须铲干净，否则残留的硬块会导致新瓷砖铺贴不平整，或出现空鼓、脱落。

                        • 按需处理的基层（1 层） 基层是否铲除，取决于其完好程度和后续装修需求，并非必须铲。

                          • 什么是基层：指粘结层下方的基础墙面或地面，比如墙面的水泥砂浆找平层、地面的混凝土垫层。这里还分为 原水泥砂浆找平层：土建阶段直接与楼板浇筑结合的找平层（厚度约15-20mm），二次找平层（5-10mm）因为平整度不足而增加的二次结构
                          • 需要铲的情况：基层出现严重开裂、起砂（用手摸掉灰）、空鼓，或厚度不足无法满足新瓷砖铺贴需求（如地面抬高高度不够）时，需要铲除后重新做基层。一般 厨房，卫生间 需要拆除到墙体
                          • 无需铲的情况：基层表面平整、结实，无破损或起砂，仅需简单打磨、清理灰尘后，即可直接在上面做新的粘结层铺贴瓷砖。
                          • 地暖：地暖因为占层高，这里还是推荐拆除 二次找平层

                        • 特殊情况：需额外处理的层 在一些老房或特殊空间，可能会遇到额外需要处理的层。

                          • 防水层：若在卫生间、厨房拆瓷砖，原墙面或地面的防水层可能会被破坏。此时无需特意铲除残留防水层，但需在铺贴新瓷砖前，重新做完整的防水层，避免漏水。
                          • 旧腻子层：部分老房墙面贴瓷砖前，可能先刷了腻子（类似墙顶的白灰层）。这种情况下，需要先铲除腻子层，露出水泥砂浆基层，再做粘结层贴瓷砖，否则腻子层不防水、不牢固，会导致瓷砖脱落。

                        在北方,因为比较寒冷，或者一楼还有一个

                        • 保温层: 集中供暖的地面强制做保温层（挤塑板厚度≥20mm），防止热量向下渗透到楼板/土壤，提升供暖效率——这两点是上海地面（无集中供暖+冬季气温极少低于0℃）完全不需要的设计。因此拆除需要额外考虑

                        木地板#

                        • 必拆的表层与辅助层（2-3 层） 这几层是拆旧地板的核心目标，必须彻底清除，为新地板铺设创造基础条件。

                          • 地板面层：最外层的旧地板本身，无论是实木地板、复合地板还是强化地板，都需要整块或分片拆除。
                          • 防潮膜 / 地垫：地板下方通常会铺一层防潮膜（塑料膜）或地垫（如泡沫地垫），用于防潮、隔音。这一层需全部撕掉或铲除，避免残留影响新防潮层效果，或产生异味、霉变。
                          • 地板龙骨（仅实木地板有）：若原地板是实木地板，下方会有木质龙骨（木条框架）。龙骨需整体拆除，且要清理掉固定龙骨的钉子或膨胀螺丝，防止残留金属件影响新地板铺设。

                        • 按需处理的基层（1 层） 基层是否铲除，取决于其完好程度和新地板的铺设要求，并非必须拆。

                          • 什么是基层：指地板辅助层下方的基础地面，常见的有水泥砂浆找平层、自流平层或原混凝土楼板。
                          • 需要铲的情况：基层表面严重不平整（高低差超过 3mm）、起砂（用手摸掉灰）、开裂，或新地板要求更平整的基层（如铺设薄款复合地板）时，需铲除原基层，重新做找平或自流平处理。
                          • 无需铲的情况：基层平整、结实，无起砂、开裂，且新地板的铺设方式（如悬浮式复合地板）对基层要求不高，仅需清理灰尘、打磨表面后即可直接使用。

                        • 特殊情况：需额外处理的层 在部分老房或特殊铺设场景中，可能会遇到额外需要处理的层。

                        • 旧地板胶 / 粘结剂层：若原地板是用胶水直接粘在基层上（如部分旧复合地板），拆除地板后会残留胶层。这一层必须彻底铲除（可用刮刀或除胶剂），否则会导致新地板铺贴不平整，或影响防潮效果。

                        • 木地台 / 垫高层：部分空间为了抬高地面，会在基层上做木质或水泥地台。若新地板无需抬高，或地台已腐朽、变形，需将地台整体拆除，恢复到原始基层高度。

                        如果拆完地面坑洼超过5mm，必须用水泥找平，否则新地板会空鼓！

                        其他#

                        • 铲除阳台保温层: 封阳台或者保温层有空鼓要把保温层拆除，提前问清如何收费;
                        • 卫生间和厨房，这2个空间的零碎的比较多，一般可能计价是单独的

                        墙体拆除#

                        墙体拆除，不同的墙体拆除的价格不一样，对于墙体的拆除还会导致后续增加修补费用的增加。

                        手持便携式切割（云石机/角磨机）#

                        适用场景#

                        非承重墙的拆除边界划线切割、瓷砖/石膏层/水泥砂浆层切割、小范围管线开槽、轻钢龙骨切断。然后配合锤子处理

                        工具与工艺#

                        1. 云石机（配金刚石片）：
                        • 定位：家装拆除最常用工具，灵活便携。
                        • 工艺：弹线标记 -> 接水管湿法作业 -> 刀片垂直墙面匀速推进。
                        • 深度控制：一般切割深度为3-5cm（切断饰面层及批荡层），起到“分界”作用，避免后期大锤拆除时带崩周边墙皮。
                        1. 角磨机：
                        • 定位：辅助工具，处理金属龙骨或边角修整。
                        • 注意：严禁拆除防护罩，切割金属时必须佩戴护目镜防止火花灼伤。

                        金刚石墙锯切割（静力切割）#

                        适用场景#

                        承重墙/剪力墙开门洞窗洞、混凝土梁柱截断、对切面平整度要求极高的新老墙体对接处。

                        工具与工艺#

                        1. 工具：固定轨道式液压/电动墙
                        2. 工艺流程：
                        • 定位安装：弹线确定洞口 -> 打膨胀螺栓固定轨道（轨道需平行于切割线）。
                        • 分步切割：安装锯片 -> 开启水冷 -> 先切竖向，后切横向（防止上方混凝土块提前下坠夹锯）。
                        • 分块移除：对于大门洞，需将内部混凝土分块切割（如切成30cm×30cm小块），便于搬运。

                        优势#

                        • 切断钢筋：金刚石锯片可平滑切断混凝土内部钢筋，切口如镜面般平整。
                        • 零震动：静力作业，不会对老旧房屋结构产生内伤。

                        水钻（金刚石钻孔机）#

                        —— 点状/线性切割，适配管线预留及“排孔法”

                        适用场景#

                        • 常规：水管/线管/空调孔/燃气孔预留。
                        • 特殊：“排孔切割法”——在无法使用墙锯的情况下，通过连续钻孔将混凝土墙体“切”下来。

                        工具与工艺#

                        1. 工具：手持或台式水钻，钻头φ25mm-φ300mm。
                        2. 工艺流程：
                        • 单孔：定位 -> 固定底座（台式） -> 湿法钻进。
                        • 排孔：沿拆除线高密度钻孔（孔间距<1cm），使墙体与主体结构分离，再剔除连接处。

                        优势#

                        • 精准避险：相比墙锯，水钻更容易感知钢筋位置，对于需保留主筋的场景（如梁上开孔），遇钢筋可及时停机调整。

                        施工组合方案与安全红线#

                        黄金组合工艺#

                        1. 标准非承重墙拆除：云石机切出四周边界（防崩边） -> 大锤/电镐拆除中间砌块。
                        2. 承重墙开门洞（标准流程）：搭设临时支撑（卸荷） -> 水钻钻四角定位孔（防过切） -> 墙锯切割四边 -> 分块移出 -> 碳纤维/型钢加固。

                        清运#

                        清运包含两个部分

                        1. 装修工人将垃圾运输到指定地点：大部分都是只包含 50m 距离，这里需要先沟通好。大多数的物业会要求堆积到固定地方，如果超过 50 m，那么是需要额外加价的。
                        2. 环卫垃圾车将垃圾清理干净：这里主要和物业对接，上海随申办有专门的 APP 可以直接申请价格比物业便宜。也可以和物业沟通
