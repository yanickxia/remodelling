                  拆旧阶段#

                  在拆旧阶段，我们需要先了解一下我们的房屋结构，这会帮助我们更好地了解施工报价。可以再复习一下 基础分层

                  基本拆除#

                  对于旧屋有一些比如床，桌子，既有的柜子等，会有一个旧房全屋拆除的步骤，会将家里的大部分物件进行拆解，然后打包好进行清运。

                  墙面拆除#

                  铲墙深度每层的价格差异很大，只铲掉饰面层重刷乳胶漆最便宜，铲到腻子层其次，铲到砂浆层价格最高，要根据自家墙面老化情况选择铲墙深度，避免被装修方过度加价。

                  对于老旧小区，一般需要拆到墙体，对于新房，一般需要拆到砂浆层。

                  注意：拆掉的越多，后面新建的越多，尽量减少拆除的内容。

                  在此阶段需要注意的事项是

                  • 拆除部分物件是可以回收的，需要约定好归属，比如门窗等。
                  • 拆除谈全包价，一口价，拆除+清运，清运费是常见的后期增项。
                  • 报价可以多参考几家，选择一个比较适合的价格。
                  • 对于外墙面还有外墙涂料/外墙漆，这部分拆除的价格会贵一些。

                  顶面拆除#

                  墙顶和墙面的结构是相似的，拆除时需要注意的事项是一般都是拆到砂浆，一般口径里面的 墙皮 包含了 腻子 + 饰面。

                  地面拆除#

                  瓷砖地面#

                  拆旧瓷砖时，核心是铲除瓷砖本身及瓷砖下方的粘结层，部分情况还需处理基层，通常需要涉及 3 层左右，具体取决于原瓷砖的铺贴工艺。

                  • 必铲的核心层（2 层） 这两层是拆旧瓷砖的主要目标，必须彻底清除，否则会影响新瓷砖铺贴质量。

                    • 瓷砖面层：最外层的旧瓷砖本身，是视觉可见的部分，需要用工具将其从墙面或地面剥离。
                    • 瓷砖粘结层：瓷砖与基层之间的粘结材料层，常见的有水泥砂浆层或瓷砖胶层。这一层必须铲干净，否则残留的硬块会导致新瓷砖铺贴不平整，或出现空鼓、脱落。

                  • 按需处理的基层（1 层） 基层是否铲除，取决于其完好程度和后续装修需求，并非必须铲。

                    • 什么是基层：指粘结层下方的基础墙面或地面，比如墙面的水泥砂浆找平层、地面的混凝土垫层。这里还分为 原水泥砂浆找平层：土建阶段直接与楼板浇筑结合的找平层（厚度约15-20mm），二次找平层（5-10mm）因为平整度不足而增加的二次结构
                    • 需要铲的情况：基层出现严重开裂、起砂（用手摸掉灰）、空鼓，或厚度不足无法满足新瓷砖铺贴需求（如地面抬高高度不够）时，需要铲除后重新做基层。一般 厨房，卫生间 需要拆除到墙体
                    • 无需铲的情况：基层表面平整、结实，无破损或起砂，仅需简单打磨、清理灰尘后，即可直接在上面做新的粘结层铺贴瓷砖。
                    • 地暖：地暖因为占层高，这里还是推荐拆除 二次找平层

                  • 特殊情况：需额外处理的层 在一些老房或特殊空间，可能会遇到额外需要处理的层。

                    • 防水层：若在卫生间、厨房拆瓷砖，原墙面或地面的防水层可能会被破坏。此时无需特意铲除残留防水层，但需在铺贴新瓷砖前，重新做完整的防水层，避免漏水。
                    • 旧腻子层：部分老房墙面贴瓷砖前，可能先刷了腻子（类似墙顶的白灰层）。这种情况下，需要先铲除腻子层，露出水泥砂浆基层，再做粘结层贴瓷砖，否则腻子层不防水、不牢固，会导致瓷砖脱落。

                  在北方,因为比较寒冷，或者一楼还有一个

                  • 保温层: 集中供暖的地面强制做保温层（挤塑板厚度≥20mm），防止热量向下渗透到楼板/土壤，提升供暖效率——这两点是上海地面（无集中供暖+冬季气温极少低于0℃）完全不需要的设计。因此拆除需要额外考虑

                  木地板#

                  • 必拆的表层与辅助层（2-3 层） 这几层是拆旧地板的核心目标，必须彻底清除，为新地板铺设创造基础条件。

                    • 地板面层：最外层的旧地板本身，无论是实木地板、复合地板还是强化地板，都需要整块或分片拆除。
                    • 防潮膜 / 地垫：地板下方通常会铺一层防潮膜（塑料膜）或地垫（如泡沫地垫），用于防潮、隔音。这一层需全部撕掉或铲除，避免残留影响新防潮层效果，或产生异味、霉变。
                    • 地板龙骨（仅实木地板有）：若原地板是实木地板，下方会有木质龙骨（木条框架）。龙骨需整体拆除，且要清理掉固定龙骨的钉子或膨胀螺丝，防止残留金属件影响新地板铺设。

                  • 按需处理的基层（1 层） 基层是否铲除，取决于其完好程度和新地板的铺设要求，并非必须拆。

                    • 什么是基层：指地板辅助层下方的基础地面，常见的有水泥砂浆找平层、自流平层或原混凝土楼板。
                    • 需要铲的情况：基层表面严重不平整（高低差超过 3mm）、起砂（用手摸掉灰）、开裂，或新地板要求更平整的基层（如铺设薄款复合地板）时，需铲除原基层，重新做找平或自流平处理。
                    • 无需铲的情况：基层平整、结实，无起砂、开裂，且新地板的铺设方式（如悬浮式复合地板）对基层要求不高，仅需清理灰尘、打磨表面后即可直接使用。

                  • 特殊情况：需额外处理的层 在部分老房或特殊铺设场景中，可能会遇到额外需要处理的层。

                  • 旧地板胶 / 粘结剂层：若原地板是用胶水直接粘在基层上（如部分旧复合地板），拆除地板后会残留胶层。这一层必须彻底铲除（可用刮刀或除胶剂），否则会导致新地板铺贴不平整，或影响防潮效果。

                  • 木地台 / 垫高层：部分空间为了抬高地面，会在基层上做木质或水泥地台。若新地板无需抬高，或地台已腐朽、变形，需将地台整体拆除，恢复到原始基层高度。

                  如果拆完地面坑洼超过5mm，必须用水泥找平，否则新地板会空鼓！

                  其他#

                  • 铲除阳台保温层: 封阳台或者保温层有空鼓要把保温层拆除，提前问清如何收费;
                  • 卫生间和厨房，这2个空间的零碎的比较多，一般可能计价是单独的

                  墙体拆除#

                  墙体拆除，不同的墙体拆除的价格不一样，对于墙体的拆除还会导致后续增加修补费用的增加。

                  清运#

                  清运包含两个部分

                  1. 装修工人将垃圾运输到指定地点：大部分都是只包含 50m 距离，这里需要先沟通好。大多数的物业会要求堆积到固定地方，如果超过 50 m，那么是需要额外加价的。
                  2. 环卫垃圾车将垃圾清理干净：这里主要和物业对接，上海随申办有专门的 APP 可以直接申请价格比物业便宜。也可以和物业沟通
